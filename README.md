# SQL Week 2 Assignment

## Overview
This repository contains SQL scripts demonstrating various SQL functions and operations as part of the Week 2 assignment. The project covers essential SQL concepts including string manipulation, numerical operations, date/time handling, joins, window functions, and set operators.

## Project Structure
The assignment is divided into the following parts:

- **part_1_string_functions.sql**: Demonstrates SQL string functions such as CONCAT, SUBSTRING, UPPER, LOWER, LENGTH, and REPLACE.
- **part_2_number_functions.sql**: Covers numerical functions including ROUND, CEIL, FLOOR, ABS, POWER, and SQRT.
- **part3_datetime.sql**: Explores date and time functions like NOW(), DATE(), YEAR(), MONTH(), DAY(), DATE_ADD(), and DATE_SUB().
- **part4_joins.sql**: Illustrates different types of SQL joins including INNER JOIN, LEFT JOIN, RIGHT JOIN, and FULL OUTER JOIN.
- **part_5_window_functions.sql**: Shows the use of window functions such as ROW_NUMBER(), RANK(), DENSE_RANK(), LAG(), LEAD(), and aggregate functions with OVER clause.
- **part_6_set_operators.sql**: Demonstrates set operations including UNION, UNION ALL, INTERSECT, and EXCEPT.

## Prerequisites
To run these SQL scripts, you will need:
- A SQL database management system (e.g., MySQL, PostgreSQL, SQL Server, or SQLite)
- A database client or command-line tool to execute the scripts

## How to Use
1. Set up a database in your preferred SQL DBMS.
2. Create the necessary tables and sample data as required by each script (if not included).
3. Execute the SQL files in order using your database client:
   - Connect to your database
   - Run each `.sql` file sequentially
4. Review the output and results in your database client.

## Notes
- Each script is self-contained and focuses on specific SQL concepts.
- Ensure your database supports the SQL standard used in these scripts.
- Some functions may vary slightly between different SQL dialects (MySQL, PostgreSQL, etc.).
